**Harikrishna Sahu, Akhlak Mahmood, Labeeba B. Shafique & Rampi Ramprasad**, [From Corpus to Innovation: Advancing Organic Solar Cell Design with Large Language Models](), submitted, 2025.

### Dataset Description

The dataset contains the following columns:

1. **PCE** (%) — Power conversion efficiency of the solar cells  
2. **Jsc** (mA/cm²) — Short-circuit current  
3. **Voc** (V) — Open-circuit voltage  
4. **FF** — Fill factor  
5. **smiles_donor** — SMILES of the donor molecule/polymer  
6. **smiles_acceptor** — SMILES of the acceptor molecule/polymer  
7. **smiles_canonical** — Canonical SMILES of donor and acceptor separated by a dot  
8. **Inverted?** — Indicates if the architecture of the organic solar cell is inverted (`True`/`False`)  
9. **Solvent** — Name of the solvent used for processing  
10. **DIO?** — Indicates if 1,8-diiodooctane (DIO) was used as an additive  
11. **Processing_Technique** — Processing technique used for preparing the thin film  
12. **TA?** — Indicates if thermal annealing was performed  
13. **SVA?** — Indicates if solvent vapor annealing was performed  
14. **TA** — Conditions for thermal annealing  
15. **SVA** — Conditions for solvent vapor annealing  
16. **thickness_nm** (nm) — Thickness of the thin film  
17. **blend_ratio** — Blend ratio between donor and acceptor  
18. **Molecular_Weight** (g/mol) — Molecular weight of the polymer used as donor molecule  
19. **DOI** — Reference (Digital Object Identifier)
