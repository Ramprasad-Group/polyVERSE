The dataset is in the following format:

Compositions: all available compositions from materials project and experimental literature  

Load: For DFT derived hardness, load is 0, for experimental hardness, load is as measured  

Hardness: DFT derived or experimentally measure  

Source: if DFT derived, then the source model else experiments  

category: five empirical models [1, 2, 3, 4, 5] and experimental [0]  

type: availability of data, [only dft], [overlapped](both DFT and exp value available), and [only_exp]
